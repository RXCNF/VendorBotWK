# Logo Waskita

## Instruksi

1. **Siapkan file logo** dengan ukuran **200x200 pixel**
   - Format yang didukung: PNG, JPG, SVG
   - Nama file: `waskita-logo.png` (atau sesuai format file Anda)

2. **Letakkan file logo** di folder ini:
   ```
   assets/images/waskita-logo.png
   ```

3. **Jika menggunakan format lain** (JPG, SVG), ubah path di `index.html`:
   - Buka file `index.html`
   - Cari: `src="assets/images/waskita-logo.png"`
   - Ganti dengan nama file Anda, contoh:
     - `src="assets/images/waskita-logo.jpg"` (untuk JPG)
     - `src="assets/images/waskita-logo.svg"` (untuk SVG)

## Spesifikasi Logo

- **Ukuran**: 200x200 pixel (disarankan)
- **Format**: PNG (transparan), JPG, atau SVG
- **Background**: Transparan (disarankan untuk PNG)
- **Aspect Ratio**: 1:1 (persegi) atau sesuai kebutuhan

## Catatan

- Logo akan otomatis di-resize untuk mobile (w-12 h-12) dan desktop (w-16 h-16)
- Maksimal ukuran: 200x200 pixel
- Logo akan tetap mempertahankan aspect ratio
